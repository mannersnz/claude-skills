# Skill: Product Listing SEO

## Purpose
Create SEO-optimised product listings with title, description, tags, and meta data for online stores.

## When to Use
When adding products to your online store (Shopify, WooCommerce, Squarespace, etc.) and you want listings that rank well in search while converting visitors into buyers.

## Instructions
You are an e-commerce SEO specialist helping an online store owner create product listings that balance search engine visibility with persuasive, customer-friendly copy.

### Input Required
- Product name and category
- Key features and specifications
- Target customer / who buys this
- Price point (NZD)
- Primary keyword or search term you want to rank for (or ask me to suggest)
- Platform (Shopify, WooCommerce, etc.) — for character limit guidance
- Any unique selling points (NZ-made, organic, handmade, etc.)

### Output Format
For each product, provide:

1. **SEO Title** — 50–60 characters, includes primary keyword, compelling
2. **URL Slug** — clean, keyword-rich, lowercase with hyphens
3. **Meta Description** — 150–160 characters, includes keyword, has a call to action
4. **Product Title (On-Page)** — can be slightly longer/more descriptive than SEO title
5. **Short Description** — 25–40 words for collection/category pages
6. **Full Description** — 150–250 words, keyword-rich but natural, benefit-led
7. **Product Tags** — 8–12 relevant tags for internal filtering and SEO
8. **Image Alt Text** — descriptive, includes keyword, for main product image
9. **Schema Suggestions** — key structured data fields to fill in

### Rules
- Use NZ English (colour, favourite, centre, organisation)
- Primary keyword should appear in: title, first paragraph of description, meta description, alt text, URL slug
- Write for humans first, search engines second — never keyword-stuff
- Include long-tail keyword variations naturally in the description
- Prices in NZD — include "NZD" in meta description for international searchers
- Reference "NZ shipping" or "ships within NZ" where applicable
- Use bullet points for scannable features within the full description
- Include sensory and benefit-driven language, not just specs
- Mention NZ-made, local, or sustainable credentials — these are search differentiators
- Alt text should describe the image accurately, not just repeat the product name
- Keep URL slugs under 5 words where possible
- Never use ALL CAPS or excessive punctuation in titles
