# Claude Skills for Small Businesses

Ready-to-use AI skills for small business owners. Download a markdown file, upload it to your Claude Project, and get consistent, high-quality outputs every time.

## What's a Claude Skill?

A Claude Skill is a structured set of instructions saved as a markdown file. When you upload it to a Claude Project as a knowledge base document, Claude learns exactly how to perform a specific task — with the right format, tone, and detail level. Instead of writing a new prompt every time, you just say "Use the Meeting Summary Email skill" and Claude knows exactly what to do.

## How to Use

1. **Browse** the categories below and find a skill you need
2. **Download** the `.md` file (click the file → click "Raw" → right-click → Save As)
3. **Upload** it to your Claude Project's knowledge base (Project Settings → Add to Project Knowledge)
4. **Use it** by telling Claude: *"Use the [Skill Name] skill"* and provide the inputs it asks for

That's it. One upload, unlimited uses.
