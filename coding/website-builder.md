# Skill: Website Builder

## Purpose
Build complete multi-page websites from scratch. Takes a business description and creates a professional site ready to deploy on Netlify, Vercel, or any static host.

## Instructions
When asked to build a website:

1. **Gather requirements:**
   - Business name and description
   - Pages needed (Home, About, Services, Contact, etc.)
   - Brand colours, logo, fonts
   - Content (or generate placeholder content based on business type)
   - Call to action (book a call, buy now, get a quote, etc.)
   - Any specific features (contact form, gallery, testimonials, pricing)

2. **Build the site:**
   - Create each page as a separate HTML file
   - Shared CSS in a `styles.css` file
   - Shared navigation and footer across all pages
   - Mobile-responsive design
   - SEO basics: meta titles, descriptions, Open Graph tags
   - Favicon placeholder

3. **Site structure:**
   ```
   /index.html          (Home)
   /about.html          (About)
   /services.html       (Services)
   /contact.html        (Contact)
   /css/styles.css      (Shared styles)
   /js/main.js          (Shared scripts)
   ```

4. **Must include:**
   - Working navigation with active page highlighting
   - Footer with business details and copyright
   - Contact form (Formspree, Netlify Forms, or mailto fallback)
   - Google Fonts loaded
   - Smooth scroll behaviour
   - Mobile hamburger menu

5. **Deployment instructions:**
   - How to preview locally (just open index.html)
   - How to deploy on Netlify (drag and drop)
   - How to connect a custom domain
